How mega-prompts work

Mega-prompts are the meta-instructions sent to Claude by the Auto Prompt Assembler. Each domain (Pre-call, Post-call, Lessons distill) has exactly one active version at a time. Editing always creates a new version; old versions are retained forever. Activating an older version rolls back without data loss.

Only one active version per domain is allowed. Activating a new version automatically deactivates the previous one.
