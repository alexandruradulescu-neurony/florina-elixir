How mega-prompts work

Mega-prompts are the meta-instructions sent to Claude by the Auto Prompt Assembler. Each domain (Pre-call, Post-call, Lessons distill) has exactly one active version at a time. Editing always creates a new version; old versions are kept forever, so activating an older one rolls back without data loss. Activating a version automatically deactivates the previous active one in the same domain.
