Manual Configuration Instructions

1. Go to ElevenLabs Dashboard

Navigate to: Developers → Webhooks

2. Add or Edit Webhook Endpoint

Click "Add Endpoint" or edit existing webhook

3. Configure Webhook

  • Set URL to: {% if webhook_url %}{{ webhook_url }}{% else %}[Your ngrok URL]/webhooks/elevenlabs/{% endif %}
  • Select event type: post_call_transcription
  • Save the webhook secret (if provided) for future signature verification

4. Save and Test

Save the webhook and test by making a call. Check the Activity Logs to verify webhook reception.

You can also use the management command: python manage.py detect_ngrok
